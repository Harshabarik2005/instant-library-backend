# Instant Library - Backend (MVP)

1. Install
   npm install

2. Copy env
   cp .env.example .env
   edit .env and set JWT_SECRET (random long string)

3. Seed DB (creates admin and student)
   npm run seed

4. Start server
   npm start
   or for dev with auto-restart:
   npm run dev

Server runs on http://localhost:4000 by default
